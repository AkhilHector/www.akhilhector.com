Mid term | GSOC 2016 | FOSSASIA
By Akhil Pandey, June 26, 2016

sTeam web interfaca has been subjected a lot of changes and below listed are some of the things which might seem interesting.

  • Added comments controller in order to integrate annotations to the web interface
  • Added workspace editor controller by supporting the ability to view and edit text files or scripts.
  • Added support for viewing PDF's, which were not functional previously.
  • Giving scope for a file permissions API.
  • Added support for the signup API, so that users can sign up using the web interface.
  • Implemented a new UI with clean and flat design.
  • Component by component implemented all of steam's web interface module's and posted them in the newUI/ directory of my repository.
  • Loading files to sTeam through ftp in order to serve my web interface.

So till the end of June concentration will be heavily invested in deploying the application later on progress will be made on dropping the old UI and sticking to the new UI.
